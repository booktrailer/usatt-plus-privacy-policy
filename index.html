Privacy Policy for USATT Plus

Effective Date: April 29, 2025

Thank you for using USATT Plus, a browser extension designed to enhance your experience on the USATT (USA Table Tennis) website by visualizing player rating data.

1. Information We Collect

USATT Plus does not collect, store, or transmit any personal data. All processing is done entirely in your browser.

  • No user names, emails, or personal identifiers are collected.
  • No browsing activity is tracked.
  • No data is sent to any external servers.

2. How We Use Data

USATT Plus reads publicly available data from the usatt.simplycompete.com website and renders graphs based on that data for visual enhancement. This data is:

  • Already visible to any site visitor.
  • Processed within the browser only.
  • Not stored or transmitted.

3. Third-Party Services

This extension does not use any third-party analytics, advertising, or tracking services.

4. Permissions

USATT Plus runs only on the usatt.simplycompete.com domain. The permission is used strictly to read and display public information for visualization purposes.

5. Data Security

Since no data is collected or transmitted, there is no risk of data breaches from this extension.

6. Changes to This Policy

If the extension changes in any way that affects data usage, we will update this privacy policy accordingly. The "Effective Date" above will always reflect the most recent update.
